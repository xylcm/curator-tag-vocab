## Non-Obvious Discoveries

1. **Figma 设计稿细节**：
   - 设计稿显示导出按钮在第二行，位置在 `left: 1098.5px` 和 `left: 1249.5px`，`top: 52px`
   - 新样式为黄色高亮：`background: #f7ff07`，`border: #d4dae0` / `rgba(238,255,184,0.69)`，`color: #007aff`
   - 按钮尺寸：高度 38px，内边距 `padding: 11px 21px`

2. **当前布局机制**：
   - `.controls` 使用 `flex-wrap: wrap`，理论上支持多行布局
   - 所有按钮当前在同一个 `.controls` 容器内，通过 `gap: 12px` 间隔
   - 控制栏是 sticky 定位（`position: sticky; top: 0`），需确保新布局不影响粘性效果

3. **样式系统**：
   - 项目使用语义化 CSS 类（`.btn-primary`, `.btn-success`, `.btn-info` 等）
   - 当前没有黄色高亮按钮的预定义类，需新增 `.btn-warning` 或 `.btn-export` 类

## Decisions & Rejected Alternatives
