# 导出按钮位置和样式调整

> GitHub Issue: #87
> 创建日期: 2026-03-04
> 状态: Draft

## 概述

调整标签管理页面顶部控制栏中导出按钮（Export Protobuf 和 Export CSV）的位置和视觉样式，使其更加醒目并与其他操作按钮在视觉上形成区分。

## 当前行为

在标签管理页面顶部的控制栏中，所有按钮排列在同一行：

- 左侧：搜索框和 5 个筛选下拉框
- 右侧：4 个操作按钮依次排列
  - "Load Tags" 按钮（蓝色）
  - "Add Tag" 按钮（绿色）
  - "Export Protobuf" 按钮（蓝色）
  - "Export CSV" 按钮（蓝色）

两个导出按钮使用与 "Load Tags" 相同的蓝色样式，在视觉上不够突出。

## 期望行为

调整后的布局和样式：

**布局变化**：
- 第一行保持：搜索框、5 个筛选下拉框、"Load Tags" 按钮、"Add Tag" 按钮
- 第二行新增：两个导出按钮独立成行
  - "Export Protobuf" 按钮位于左侧
  - "Export CSV" 按钮紧邻其右侧

**样式变化**：
- 两个导出按钮采用黄色高亮样式：
  - 背景色：亮黄色（`#f7ff07`）
  - 文字颜色：蓝色（`#007aff`）
  - 边框：淡黄色系
  - 保持圆角和内边距与其他按钮一致

**视觉效果**：
- 导出按钮在视觉上更加醒目，与主要操作按钮形成区分
- 两个导出按钮在第二行独立展示，避免第一行过于拥挤
- 整体布局更加清晰，功能分组更加明确

## 验收标准

- [ ] 导出按钮从第一行移至第二行，位于控制栏下方
- [ ] "Export Protobuf" 按钮在左，"Export CSV" 按钮在右，两者水平相邻
- [ ] 两个导出按钮的背景色为亮黄色（`#f7ff07`）
- [ ] 两个导出按钮的文字颜色为蓝色（`#007aff`）
- [ ] 导出按钮的圆角、内边距、字体大小与其他按钮保持一致
- [ ] 鼠标悬停时有适当的视觉反馈（如背景色微调）
- [ ] 在不同屏幕宽度下布局正常，无错位或重叠
- [ ] 所有原有功能（点击导出、筛选、加载等）正常工作
