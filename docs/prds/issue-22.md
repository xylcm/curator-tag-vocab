# 导出按钮背景颜色修改

> GitHub Issue: #22
> 创建日期: 2025-02-11
> 状态: Draft

## 概述

将标签管理页面中导出按钮（Export Protobuf、Export CSV）的背景颜色从蓝色改为白色，使界面视觉更加协调统一。

## 当前行为

导出按钮（Export Protobuf 和 Export CSV）当前使用 `.btn-info` 样式类，背景色为浅蓝色（#e3f2ff），文字颜色为蓝色（#007aff）。在当前的界面设计中，蓝色背景与整体色调不够协调。

## 期望行为

导出按钮背景颜色改为白色，具体颜色值如下：

| 状态 | 背景色 | 文字色 | 边框色 |
|------|--------|--------|--------|
| 默认状态 | `#ffffff`（白色） | `#007aff`（蓝色） | `#b8dcff`（浅蓝色） |
| 悬停状态 | `#f5f5f5`（浅灰白） | `#007aff`（蓝色） | `#9fceff`（浅蓝色） |

与页面其他次要操作按钮保持一致的视觉风格，提升界面整体协调性。

## 验收标准

- [ ] Export Protobuf 按钮背景颜色为白色（`#ffffff`）
- [ ] Export CSV 按钮背景颜色为白色（`#ffffff`）
- [ ] 按钮文字颜色保持蓝色（`#007aff`）
- [ ] 按钮边框颜色为浅蓝色（`#b8dcff`）
- [ ] 悬停状态背景色为浅灰白（`#f5f5f5`）
- [ ] 悬停状态边框色为浅蓝色（`#9fceff`）
