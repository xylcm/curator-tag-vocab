# 导出按钮背景色修改

> GitHub Issue: #33
> 创建日期: 2026-02-12
> 状态: Draft

## 概述

将标签管理界面中「Export Protobuf」和「Export CSV」两个导出按钮的背景色从蓝色修改为红色，以提升视觉辨识度。

## 当前行为

在标签管理页面的控制栏中，有两个导出按钮：
- 「Export Protobuf」按钮
- 「Export CSV」按钮

这两个按钮当前使用蓝色主题（`.btn-info` 样式）：
- 背景色：浅蓝色（#e3f2ff）
- 文字颜色：蓝色（#007aff）
- 边框：蓝色

## 期望行为

两个导出按钮应使用红色主题：
- 背景色：浅红色/粉色（#ffebe9）
- 文字颜色：红色（#ff3b30）
- 边框：红色系

悬停时保持相应的红色系配色变化。

## 验收标准

- [ ] 「Export Protobuf」按钮显示为红色背景样式
- [ ] 「Export CSV」按钮显示为红色背景样式
- [ ] 按钮悬停状态保持红色系配色
- [ ] 其他按钮（Load Tags、Add Tag）的样式保持不变
