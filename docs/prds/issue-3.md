# 导出按钮背景色改为浅色

> GitHub Issue: #3
> 创建日期: 2025-01-20
> 状态: Draft

## 概述
将标签管理页面中导出按钮（Export Protobuf 和 Export CSV）的背景色从当前的浅蓝色改为更浅的色调，以优化视觉层次和界面一致性。

## 当前行为
- 导出按钮使用 `btn-info` 样式类
- 背景色为 `#e3f2ff`（浅蓝色）
- 文字颜色为 `#007aff`（蓝色）
- 悬停时背景色变为 `#d0e8ff`

## 期望行为
- 导出按钮背景色改为更浅的色调
- 建议改为白色背景（`#ffffff`）或极浅灰色（`#f5f5f7`）
- 保持蓝色文字（`#007aff`）以维持可识别性
- 悬停状态相应调整

## 验收标准
- [ ] "Export Protobuf" 按钮背景色显示为浅色（白色或极浅灰）
- [ ] "Export CSV" 按钮背景色显示为浅色（白色或极浅灰）
- [ ] 按钮文字保持蓝色（`#007aff`）
- [ ] 悬停状态视觉反馈正常
- [ ] 按钮与其他控件（Load Tags、Add Tag）的视觉层次清晰可辨
