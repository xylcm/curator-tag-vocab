# 导出按钮背景色调整

> GitHub Issue: #67
> 创建日期: 2026-02-27
> 状态: Draft

## 概述
将导出按钮（Export Protobuf 和 Export CSV）的背景色从当前的浅蓝色调整为浅白色，使界面视觉风格更加统一协调。

## 当前行为
- 导出按钮（Export Protobuf、Export CSV）使用 `.btn-info` 样式类
- 当前背景色为浅蓝色（`#e3f2ff`），与页面整体浅色调形成一定视觉反差
- 按钮文字颜色为蓝色（`#007aff`）

## 期望行为
- 导出按钮背景色调整为浅白色（`#f5f5f5` 或相近的浅灰色调）
- 保持按钮文字颜色不变（`#007aff`）
- 保持按钮边框和悬停效果的一致性
- 与页面上其他辅助操作按钮（如分页按钮）的视觉风格保持一致

## 验收标准
- [ ] Export Protobuf 按钮背景色显示为浅白色
- [ ] Export CSV 按钮背景色显示为浅白色
- [ ] 按钮文字颜色保持蓝色（`#007aff`）
- [ ] 按钮悬停效果正常工作
- [ ] 按钮边框样式与页面其他按钮保持一致
