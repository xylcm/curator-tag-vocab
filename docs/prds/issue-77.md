# 导出按钮背景色调整

> GitHub Issue: #77
> 创建日期: 2025-01-21
> 状态: Draft

## 1. 概述

将标签管理页面中导出功能按钮的背景色从浅蓝色调整为白色，使界面视觉风格更加统一、简洁。

## 2. 用户故事

- 作为标签管理系统的用户，我希望导出按钮使用白色背景，以便与页面整体简洁风格保持一致。

## 3. 功能描述

### 3.1 入口与触发方式

导出按钮位于标签管理页面的控制栏区域，用户访问 `/tagging/vocab` 页面即可看到。

### 3.2 界面与交互

**当前状态：**
- 导出按钮（"Export Protobuf" 和 "Export CSV"）使用 `.btn-info` 样式类
- 当前背景色为浅蓝色（`#e3f2ff`），边框为 `#b8dcff`，文字为蓝色（`#007aff`）

**期望状态：**
- 导出按钮背景色改为白色（`#ffffff`）
- 保持边框和文字颜色不变，确保按钮仍可识别为可点击元素
- 悬停状态背景色调整为 `#f5f5f5`，保持交互反馈

**涉及元素：**
- "Export Protobuf" 按钮（ID: `export-protobuf-btn`）
- "Export CSV" 按钮（ID: `export-csv-btn`）

## 4. 验收标准

- [ ] "Export Protobuf" 按钮背景色显示为白色
- [ ] "Export CSV" 按钮背景色显示为白色
- [ ] 按钮边框保持蓝色（`#b8dcff`）
- [ ] 按钮文字保持蓝色（`#007aff`）
- [ ] 鼠标悬停时按钮背景色变为浅灰色（`#f5f5f5`）
- [ ] 其他按钮（Load Tags、Add Tag）样式保持不变
