# 导出按钮布局调整

> GitHub Issue: #37
> 创建日期: 2025-02-12
> 状态: Draft

## 概述
将导出按钮（Export Protobuf 和 Export CSV）从当前控制栏区域移动到页面右上角，与页面标题水平对齐，使导出功能更加醒目且符合用户操作习惯。

## 当前行为
- 导出按钮（"Export Protobuf" 和 "Export CSV"）位于页面中部的控制栏（`.controls`）内
- 与搜索框、筛选器、排序选项等控件混杂在同一行
- 导出功能不够突出，用户需要在中部区域寻找

## 期望行为
- 导出按钮移动到页面右上角区域
- 与页面标题 "Curator Tag Vocabulary" 保持水平对齐
- 按钮靠右展示，与标题形成左右布局
- 按钮样式保持不变，保持视觉一致性

## 验收标准
- [ ] 导出按钮（Export Protobuf 和 Export CSV）显示在页面右上角
