# 导出按钮背景色调整

> GitHub Issue: #81
> 创建日期: 2026-03-02
> 状态: Draft

## 概述

将标签管理界面顶部控制栏中的两个导出按钮（Export Protobuf 和 Export CSV）的背景色从当前的浅蓝色改为白色，以优化视觉层次和界面一致性。

## 当前行为

用户在标签管理界面（`/tagging/vocab`）顶部控制栏中看到两个导出按钮：
- "Export Protobuf" 按钮
- "Export CSV" 按钮

这两个按钮当前使用浅蓝色背景（`#e3f2ff`），与 "Load Tags" 按钮的视觉样式相同，在视觉上与其他操作按钮（如绿色的 "Add Tag" 按钮）形成对比。

## 期望行为

变更后，两个导出按钮的背景色应改为白色（`#ffffff`），同时保持：
- 按钮边框和文字颜色保持清晰可见
- 鼠标悬停时有明显的视觉反馈
- 按钮点击后的导出功能不受影响
- 与界面整体设计风格协调

用户在使用导出功能时，视觉体验更加统一，导出按钮在视觉层次上更加低调，突出主要操作按钮（Load Tags、Add Tag）。

## 验收标准

- [ ] Export Protobuf 按钮的背景色为白色
- [ ] Export CSV 按钮的背景色为白色
- [ ] 按钮文字清晰可读，对比度符合可访问性标准
- [ ] 鼠标悬停时有明显的视觉反馈效果
- [ ] 按钮点击后导出功能正常工作（Protobuf 和 CSV 格式均可正常下载）
- [ ] 在不同浏览器（Chrome、Firefox、Safari）中显示一致
- [ ] 响应式布局下（移动端、平板）按钮样式正常显示
