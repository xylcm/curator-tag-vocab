# 导出按钮位置和样式调整

> GitHub Issue: #95
> 创建日期: 2026-03-05
> 状态: Draft

## 1. 概述

调整标签管理页面中导出按钮的位置和样式，使其与主操作按钮（Load Tags、Add Tag）分离，形成独立的第二行布局，并应用醒目的黄色背景样式，提升界面视觉层次和操作区分度。

## 2. 用户故事

- 作为标签管理员，我希望导出按钮与常规操作按钮在视觉上区分开，以便快速识别数据导出功能
- 作为标签管理员，我希望导出按钮以醒目的颜色呈现，以便在需要导出数据时快速定位

## 3. 功能描述

### 3.1 入口与触发方式

导出按钮位于标签管理页面的控制栏区域，与筛选器、搜索框等控件一同展示。

### 3.2 核心流程

1. 用户进入标签管理页面
2. 在控制栏区域看到第一行操作按钮：Load Tags（加载标签）、Add Tag（添加标签）
3. 在控制栏第二行看到导出按钮：Export Protobuf、Export CSV
4. 点击导出按钮后，系统导出对应格式的数据文件

### 3.3 界面与交互

**位置调整：**
- 导出按钮（Export Protobuf、Export CSV）从第一行移至第二行
- 第一行保留：搜索框、筛选下拉框、Load Tags 按钮、Add Tag 按钮
- 第二行专门用于放置导出按钮

**样式调整：**
- Export Protobuf 按钮：
  - 背景色：#f7ff07（亮黄色）
  - 边框：1px solid #d4dae0
  - 文字颜色：#007aff（蓝色）
  - 圆角：6px
  - 高度：38px
  - 内边距：10px 21px

- Export CSV 按钮：
  - 背景色：#f7ff07（亮黄色）
  - 边框：1px solid rgba(238, 255, 184, 0.69)
  - 文字颜色：#007aff（蓝色）
  - 圆角：6px
  - 高度：38px
  - 内边距：10px 21px

**布局细节：**
- 第二行按钮右对齐，与第一行按钮区域对齐
- 按钮间距保持 12px
- 整体控制栏保持原有背景色 #f5f5f7 和底部边框

## 4. 验收标准

- [ ] Export Protobuf 和 Export CSV 按钮显示在控制栏第二行
- [ ] 第一行仅显示搜索框、筛选器、Load Tags 和 Add Tag 按钮
- [ ] 导出按钮背景色为亮黄色（#f7ff07）
- [ ] 导出按钮文字颜色为蓝色（#007aff）
- [ ] 导出按钮样式与 Figma 设计稿一致
- [ ] 导出按钮功能保持不变，点击后正常导出数据
- [ ] 响应式布局在各种屏幕尺寸下正常显示
