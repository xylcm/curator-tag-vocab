# 导出按钮背景色浅色化

> GitHub Issue: #63
> 创建日期: 2025-02-27
> 状态: Draft

## 概述
将页面顶部导出按钮（Export Protobuf、Export CSV）的背景色从当前的蓝色改为浅色，使界面视觉层次更加协调，导出功能与主要操作按钮（Load Tags、Add Tag）区分开来。

## 当前行为
- "Export Protobuf" 和 "Export CSV" 按钮使用 `btn-info` 样式
- 按钮背景色为浅蓝色（#e3f2ff），文字为蓝色（#007aff）
- 视觉上与 "Load Tags" 按钮（同为蓝色系）较为接近

## 期望行为
- 导出按钮背景色改为浅灰色系
- 保持按钮文字颜色为深灰色，确保可读性
- 与主操作按钮形成视觉区分，突出主要功能

## 验收标准
- [ ] "Export Protobuf" 按钮背景色为浅灰色（如 #f5f5f5 或 #f0f0f0）
- [ ] "Export CSV" 按钮背景色与 Export Protobuf 保持一致
- [ ] 按钮文字颜色为深灰色（如 #6e6e73 或 #1d1d1f），确保对比度符合可读性标准
- [ ] 按钮悬停状态有适当的视觉反馈
- [ ] 按钮边框样式与其他按钮保持一致
