# 导出按钮字体颜色修改为绿色

> GitHub Issue: #5
> 创建日期: 2026-02-10
> 状态: Draft

## 概述
将标签管理页面中两个导出按钮（"Export Protobuf" 和 "Export CSV"）的字体颜色从蓝色改为绿色，使其与 "Add Tag" 按钮的视觉风格保持一致。

## 当前行为
在标签管理页面的控制栏区域，有两个导出按钮：
- "Export Protobuf" 按钮
- "Export CSV" 按钮

这两个按钮当前使用 `.btn-info` 样式类，显示为蓝色字体（#007aff），与 "Load Tags" 按钮颜色相同。

## 期望行为
两个导出按钮的字体颜色应改为绿色（#34c759），与 "Add Tag" 按钮保持一致，使用 `.btn-success` 样式类。

## 验收标准
- [ ] "Export Protobuf" 按钮的字体颜色显示为绿色（#34c759）
- [ ] "Export CSV" 按钮的字体颜色显示为绿色（#34c759）
- [ ] 按钮的其他样式（背景色、边框、悬停效果）保持不变
