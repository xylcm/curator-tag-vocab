# 导出按钮背景色变更为紫色

> GitHub Issue: #79
> 创建日期: 2026-02-28
> 状态: Draft

## 概述
将标签管理页面中两个导出按钮（"Export Protobuf" 和 "Export CSV"）的背景色从当前的蓝色改为紫色，以提升视觉区分度和用户体验。

## 当前行为
在标签管理页面的控制栏中，有两个导出按钮：
- "Export Protobuf" 按钮
- "Export CSV" 按钮

这两个按钮当前使用 `.btn-info` 样式类，显示为浅蓝色背景（#e3f2ff）和蓝色文字（#007aff）。

## 期望行为
两个导出按钮的背景色应改为紫色，使其在视觉上与其他操作按钮（如蓝色的 "Load Tags" 和绿色的 "Add Tag"）形成区分，同时保持按钮的交互状态（hover、active）样式一致。

## 验收标准
- [ ] "Export Protobuf" 按钮的背景色显示为紫色
- [ ] "Export CSV" 按钮的背景色显示为紫色
- [ ] 按钮的 hover 状态保持正常交互反馈
- [ ] 按钮的 active/点击状态保持正常交互反馈
- [ ] 按钮文字颜色与紫色背景保持良好的可读性对比
