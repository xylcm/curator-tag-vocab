# 导出按钮背景色修改

> GitHub Issue: #69
> 创建日期: 2026-02-28
> 状态: Draft

## 概述
将标签管理页面中「Export Protobuf」和「Export CSV」两个导出按钮的背景色从浅蓝色修改为白色，使按钮样式与整体界面风格更协调。

## 当前行为
在标签管理页面（`/tagging/vocab`）的顶部控制栏中，有两个导出按钮：
- Export Protobuf
- Export CSV

这两个按钮当前使用 `btn-info` 样式，背景色为浅蓝色（`#e3f2ff`），文字颜色为蓝色（`#007aff`）。

## 期望行为
导出按钮的背景色改为白色（`#ffffff`），保持蓝色文字和边框，使按钮在视觉上更加简洁，与其他白色背景的表单元素（如输入框、下拉选择框）保持一致。

## 验收标准
- [ ] 「Export Protobuf」按钮背景色为白色
- [ ] 「Export CSV」按钮背景色为白色
- [ ] 按钮文字颜色保持蓝色（`#007aff`）
- [ ] 按钮边框保持蓝色系样式
- [ ] 鼠标悬停时按钮有适当的视觉反馈
