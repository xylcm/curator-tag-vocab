# 导出按钮背景颜色修改

> GitHub Issue: #11
> 创建日期: 2025-02-10
> 状态: Draft

## 概述
将导出按钮（Export Protobuf 和 Export CSV）的背景颜色修改为浅蓝色，以提升视觉识别度和用户体验。

## 当前行为
导出按钮当前使用 `.btn-info` 样式类，背景色为 `#e3f2ff`（浅蓝色），文字颜色为 `#007aff`（蓝色）。

## 期望行为
导出按钮的背景颜色应使用更明显的浅蓝色，使其在界面中更加突出，便于用户识别导出功能。

## 验收标准
- [ ] Export Protobuf 按钮的背景颜色为浅蓝色
- [ ] Export CSV 按钮的背景颜色为浅蓝色
- [ ] 按钮文字颜色与背景色保持足够的对比度，确保可读性
- [ ] 按钮悬停状态颜色协调一致
