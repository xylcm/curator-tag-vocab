# 导出按钮背景色调整

> GitHub Issue: #73
> 创建日期: 2026-02-28
> 状态: Draft

## 概述
将导出按钮（Export Protobuf 和 Export CSV）的背景色统一调整为浅蓝色，使其在视觉上更加醒目且与整体界面风格协调。

## 当前行为
导出按钮（"Export Protobuf" 和 "Export CSV"）当前使用 `btn-info` 样式类，背景色为 `#e3f2ff`（浅蓝色），文字颜色为 `#007aff`（蓝色）。

## 期望行为
导出按钮的背景色调整为更浅的蓝色 `#e8f4ff`，使按钮在控制栏中更加柔和且易于识别。

## 验收标准
- [ ] "Export Protobuf" 按钮的背景色显示为浅蓝色（`#e8f4ff`）
- [ ] "Export CSV" 按钮的背景色显示为浅蓝色（`#e8f4ff`）
- [ ] 按钮 hover 状态下的背景色相应调整为 `#d6ecff`
- [ ] 按钮文字颜色保持为蓝色（`#007aff`）不变
- [ ] 按钮边框颜色保持为 `#b8dcff` 不变
