# 导出按钮背景颜色修改

> GitHub Issue: #20
> 创建日期: 2026-02-11
> 状态: Draft

## 概述

将标签词表管理页面中导出按钮（"Export Protobuf" 和 "Export CSV"）的背景颜色从蓝色修改为白色，使界面视觉更加协调。

## 当前行为

在标签词表管理页面的控制栏区域，有两个导出按钮：
- "Export Protobuf" 按钮
- "Export CSV" 按钮

这两个按钮当前使用 `btn-info` 样式类，显示为浅蓝色背景（`#e3f2ff`）和蓝色文字（`#007aff`），与页面整体风格不够协调。

## 期望行为

导出按钮的背景颜色改为白色，同时保持按钮的可识别性和交互反馈：
- 默认状态：白色背景，深色文字
- 悬停状态：轻微灰色背景，保持视觉反馈
- 保持按钮边框样式，确保按钮边界清晰可见

## 验收标准

- [ ] "Export Protobuf" 按钮背景颜色为白色
- [ ] "Export CSV" 按钮背景颜色为白色
- [ ] 按钮悬停时有明显的视觉反馈
- [ ] 按钮文字颜色与背景有足够的对比度，确保可读性
- [ ] 按钮边框保持可见，明确按钮边界
