# 导出按钮布局和样式调整

> GitHub Issue: #15
> 创建日期: 2025-01-21
> 状态: Draft

## 概述

将导出按钮（Export Protobuf、Export CSV）从当前的控制栏区域移动到页面右上角，与标题 "Curator Tag Vocabulary" 水平对齐并靠右展示，以优化页面布局和视觉层次。

## 当前行为

- 导出按钮（Export Protobuf、Export CSV）位于 `.controls` 控制栏区域内
- 控制栏位于标题和统计信息下方，包含搜索框、筛选器、排序选项和操作按钮
- 导出按钮与其他控制元素混在一起，视觉上不够突出

## 期望行为

- 导出按钮移动到页面右上角区域
- 与标题 "Curator Tag Vocabulary" 保持水平对齐
- 按钮靠右展示，与页面内容区域右边缘对齐
- 保持按钮原有的样式和交互行为

## 验收标准

- [ ] 导出按钮（Export Protobuf、Export CSV）显示在页面右上角
- [ ] 按钮与标题 "Curator Tag Vocabulary" 水平对齐
- [ ] 按钮靠右展示，与页面内容区域右边缘对齐
- [ ] 按钮保持原有的视觉样式（颜色、大小、悬停效果）
- [ ] 按钮保持原有的点击交互行为
- [ ] 响应式布局下（移动端）按钮位置合理适配
