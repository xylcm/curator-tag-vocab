# 导出按钮背景色调整

> GitHub Issue: #83
> 创建日期: 2026-03-02
> 状态: Draft

## 概述

将标签词汇管理页面顶部控制栏中的两个导出按钮（Export Protobuf 和 Export CSV）的背景色从当前的浅蓝色改为浅白色，以降低视觉权重，使其与页面整体风格更协调。

## 当前行为

用户在标签词汇管理页面（`/tagging/vocab`）顶部控制栏看到四个操作按钮：
- "Load Tags" 按钮：浅蓝色背景（`btn-primary` 样式）
- "Add Tag" 按钮：浅绿色背景（`btn-success` 样式）
- "Export Protobuf" 按钮：浅蓝色背景（`btn-info` 样式）
- "Export CSV" 按钮：浅蓝色背景（`btn-info` 样式）

两个导出按钮与 "Load Tags" 按钮颜色相同，视觉上较为突出。

## 期望行为

变更后，用户在同一位置看到的按钮样式应为：
- "Load Tags" 按钮：保持浅蓝色背景（不变）
- "Add Tag" 按钮：保持浅绿色背景（不变）
- "Export Protobuf" 按钮：浅白色/浅灰色背景，视觉权重降低
- "Export CSV" 按钮：浅白色/浅灰色背景，视觉权重降低

导出按钮的文字颜色、边框、圆角、内边距等其他样式保持不变，仅调整背景色。按钮的交互行为（点击触发导出）完全不变。

## 验收标准

- [ ] 两个导出按钮的背景色已从浅蓝色改为浅白色/浅灰色
- [ ] 导出按钮的文字清晰可读，与新背景色有足够对比度
- [ ] 导出按钮的其他样式（边框、圆角、内边距、字体）保持不变
- [ ] 导出按钮的 hover 和 active 状态样式与新背景色协调
- [ ] 其他按钮（Load Tags、Add Tag）的样式不受影响
- [ ] 导出功能正常工作，点击按钮能成功下载文件
