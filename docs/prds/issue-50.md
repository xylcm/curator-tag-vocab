# 导出按钮背景色调整

> GitHub Issue: #50
> 创建日期: 2026-02-26

## 概述
将导出按钮（Export Protobuf 和 Export CSV）的背景色从当前的浅蓝色调整为白色，以改善界面视觉一致性。

## 当前行为
导出按钮（位于页面顶部控制栏，"Add Tag" 按钮右侧）当前显示为浅蓝色背景（#e3f2ff），与整体界面风格不够协调。

## 期望行为
导出按钮的背景色应为白色（#ffffff），保持按钮边框和文字颜色不变，使界面视觉更加统一。

## 验收标准
- [ ] Export Protobuf 按钮背景色显示为白色
- [ ] Export CSV 按钮背景色显示为白色
- [ ] 按钮边框颜色保持不变
- [ ] 按钮文字颜色保持不变
- [ ] 按钮悬停状态视觉效果正常
