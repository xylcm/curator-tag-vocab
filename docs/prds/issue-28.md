# 导出按钮背景色修改

> GitHub Issue: #28
> 创建日期: 2026-02-11
> 状态: Draft

## 概述

将标签管理页面中导出按钮（Export Protobuf、Export CSV）的背景颜色从蓝色修改为白色，使界面视觉更加协调。

## 当前行为

在标签管理页面（`/tagging/vocab`）中，有两个导出按钮：
- "Export Protobuf" 按钮
- "Export CSV" 按钮

这两个按钮当前使用 `.btn-info` 样式类，背景色为浅蓝色（`#e3f2ff`），文字颜色为蓝色（`#007aff`）。

## 期望行为

导出按钮的背景色应改为白色，与页面整体风格更加协调。

## 验收标准

- [ ] "Export Protobuf" 按钮背景色为白色
- [ ] "Export CSV" 按钮背景色为白色
- [ ] 按钮文字颜色保持可读性（建议保持蓝色 `#007aff` 或调整为适合的深色）
- [ ] 按钮悬停状态有适当的视觉反馈
- [ ] 其他按钮（Load Tags、Add Tag）的样式不受影响
